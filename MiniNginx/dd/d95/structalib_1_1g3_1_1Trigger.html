Detailed Description

一个简单的触发器，使用test进行时间测试

例子:
Clock clk;
Trigger tri(clk,100);//100ms触发一次时钟
while(true){
if(tri.test()){//默认自动重置时间
std::cout << "Hello World!" << std::endl;
}
}
